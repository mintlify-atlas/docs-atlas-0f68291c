---
title: "Export and import"
description: "Export your VRCX data as CSV or JSON files, and import favorite groups or group moderation bans."
---

VRCX lets you export your data to standard file formats so you can back it up, share it, or use it outside of VRCX. You can also import certain data types to restore or migrate your configuration.

## What you can export

<CardGroup cols={2}>
  <Card title="Friends list" icon="users">
    Export your full VRChat friends list with display names and user IDs.
  </Card>
  <Card title="Avatar list" icon="person">
    Export your saved avatar list with avatar names and IDs.
  </Card>
  <Card title="Discord names" icon="discord">
    Export the Discord usernames you have linked to VRChat friends in VRCX.
  </Card>
  <Card title="Notes" icon="note-sticky">
    Export the notes and memos you have written for friends.
  </Card>
  <Card title="Favorite groups" icon="star">
    Export your VRChat favorite groups and their contents.
  </Card>
</CardGroup>

## What you can import

<CardGroup cols={2}>
  <Card title="Favorite groups" icon="star">
    Import a previously exported favorite groups file to restore or migrate your favorites.
  </Card>
  <Card title="Group moderation bans" icon="ban">
    Import a list of group moderation bans to apply them in bulk.
  </Card>
</CardGroup>

## Where to find export and import

All export and import options are in the **Tools** menu.

<Steps>
  <Step title="Open the Tools menu">
    In the VRCX menu bar, click **Tools**.
  </Step>
  <Step title="Select the export or import option">
    Choose from the available export or import actions. Each data type has its own menu entry.
  </Step>
  <Step title="Choose a file location">
    For exports, pick where to save the file. For imports, select the file you previously exported.
  </Step>
</Steps>

## File formats

| Data type | Format |
| --------- | ------ |
| Friends list | CSV |
| Avatar list | CSV |
| Discord names | CSV |
| Notes | CSV |
| Favorite groups | JSON |
| Group moderation bans | JSON (import only) |

<Note>
  Exported CSV files can be opened in any spreadsheet application such as Excel or Google Sheets. JSON files can be viewed in any text editor.
</Note>
