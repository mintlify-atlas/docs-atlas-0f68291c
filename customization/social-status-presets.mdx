---
title: "Social status presets"
description: "Save and instantly apply VRChat status and status description combinations from anywhere in VRCX."
---

Social status presets let you define a VRChat status and status description as a named pair, then switch to that combination with a single click. This is useful when you frequently change between a set of known states — for example, switching between "Join Me" while exploring worlds and "Busy" when you want to focus.

<Note>
  Presets update your actual VRChat status through the API. The change takes effect immediately and is visible to your friends, just as if you had changed your status from within VRChat.
</Note>

## Available status values

VRChat supports four status levels:

- **Join Me** — Open to anyone joining you
- **Active** — Online and active
- **Ask Me** — Available but prefer to be asked first
- **Busy** — Prefer not to be disturbed

You can pair any of these with a custom status description (the short text shown under your name).

## Creating a preset

<Steps>
  <Step title="Open the status presets panel">
    Status presets are accessible from the VRCX sidebar or from your own user dialog. Look for the presets section near your current status display.
  </Step>
  <Step title="Add a new preset">
    Click the button to add a preset. A form appears with two fields: a status selector and a description text field.
  </Step>
  <Step title="Choose a status">
    Select one of the four VRChat status levels from the dropdown: **Join Me**, **Active**, **Ask Me**, or **Busy**.
  </Step>
  <Step title="Enter a status description">
    Type the short message you want to appear as your status description when this preset is active. This field is optional — leave it blank to clear the description when the preset is applied.
  </Step>
  <Step title="Save the preset">
    Confirm the preset. It now appears in your presets list with the status and description you entered.
  </Step>
</Steps>

## Applying a preset

Click the preset's name or apply button in the list. VRCX immediately sets your VRChat status and status description to the saved values.

<Note>
  Presets are especially useful in combination with the auto state change feature in **Settings > Social**, which can automatically switch your status based on whether friends are in your instance. You can use presets for manual overrides on top of that automation.
</Note>

## Managing presets

You can edit or delete existing presets from the same panel. Presets are stored locally in VRCX and are not synced to VRChat or other devices.
