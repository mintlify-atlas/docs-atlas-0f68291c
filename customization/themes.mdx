VRCX supports CSS-based themes that let you completely restyle the application's appearance. Themes are created and shared by the community.

## How themes work

Themes are CSS files that override VRCX's default styles. When a theme is active, VRCX injects the stylesheet into the application, replacing colors, fonts, borders, and other visual properties.

<Note>
  Themes only change the appearance of VRCX itself — they have no effect on VRChat or any other application.
</Note>

## Finding themes

The community maintains a collection of themes on the VRCX GitHub wiki.

<Tip>
  Browse available themes at [github.com/vrcx-team/VRCX/wiki/Themes](https://github.com/vrcx-team/VRCX/wiki/Themes). Each theme typically includes a preview screenshot and installation instructions.
</Tip>

## Installing a theme

<Steps>
  <Step title="Download the theme file">
    Locate a theme from the community wiki or another source. Download the `.css` file to your computer.
  </Step>
  <Step title="Open Settings">
    In VRCX, open **Settings** from the sidebar and navigate to the **Interface** tab.
  </Step>
  <Step title="Load the theme">
    Find the theme option in the Interface settings and select your downloaded `.css` file. VRCX applies the theme immediately — no restart required.
  </Step>
  <Step title="Verify the result">
    The VRCX interface should now reflect the new theme. If something looks wrong, you can remove the theme and return to the default appearance from the same settings panel.
  </Step>
</Steps>

## Switching back to the default

To remove a theme and restore VRCX's default appearance, open **Settings > Interface** and clear the theme selection.

<Note>
  VRCX also has a built-in dark/light mode toggle that works independently of themes. You can switch between dark and light using the toggle in the title bar or from **Settings > Interface**.
</Note>

## Creating your own theme

Themes are standard CSS files. If you are familiar with CSS, you can create a custom theme by targeting VRCX's UI elements directly.

<Tip>
  Check the existing community themes on the wiki for examples of how selectors and variables are structured. Contributing your theme back to the wiki is a great way to share it with other users.
</Tip>
